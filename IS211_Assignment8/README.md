Read the following sections in Design Patterns in Python by Rahul Verma, Chetan Giridha:
Preface (pg 7 - 10)
Model-View-Controller Pattern (pg 10 - 15)
Observer Pattern (pg 19 - 23)
Facade Pattern (pg 23 - 27)
Factory Pattern (pg 32 - 35)
Proxy Pattern (pg 35 - 37)

Python 3 Patterns, Recipes and Idioms. A nice introduction to design patterns, and some examples in python. Note that the examples are in Python 3, but is still applicable to the topic
http://python-3-patterns-idioms-test.readthedocs.org/en/latest/PatternConcept.html
Chapter 19 of Introduction to Programming using Python has a good introduction to these design patterns in Python. Note, this text is aimed at  Biologists, but is still a useful text
